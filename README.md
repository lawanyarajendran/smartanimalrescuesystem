Smart Rescue – Animal Rescue and Volunteer Management System

Project Overview

Smart Rescue is a web-based Animal Rescue and Volunteer Management System developed using Java, Spring Boot, Spring Security, JWT Authentication, PostgreSQL, HTML, CSS, and JavaScript. The system helps users report injured or abandoned animals and enables volunteers to track and manage rescue operations through a centralized platform.

Features

- User Registration and Login
- JWT Authentication
- Animal Rescue Reporting
- Volunteer Workflow Management
- Rescue Status Tracking
- CRUD Operations
- PostgreSQL Database Integration
- REST API Communication
- Validation and Logging
- Pagination Support
- Swagger API Documentation
- Frontend–Backend Integration

Technology Stack

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate ORM
- PostgreSQL
- HTML
- CSS
- JavaScript
- Maven
- Swagger
- Postman

Rescue Workflow

User Registration → Login → Animal Reporting → Volunteer Review → Status Update (PENDING → ACCEPTED → RESCUED)
